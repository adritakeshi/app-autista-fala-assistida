# API

## Perfis
- `GET /api/profiles`
- `POST /api/profiles`
- `GET /api/profiles/:id`
- `PUT /api/profiles/:id`
- `DELETE /api/profiles/:id`

## Pranchas
- `GET /api/boards`
- `POST /api/boards`
- `GET /api/boards/:id`

## Categorias
- `POST /api/categories`
- `DELETE /api/categories/:id`

## Cartões
- `POST /api/cards`
- `PUT /api/cards/:id`
- `DELETE /api/cards/:id`

## Rotinas
- `GET /api/routines`
- `POST /api/routines`
- `PATCH /api/routines/:id`

## Relatórios
- `GET /api/reports`
- `POST /api/reports`
- `DELETE /api/reports/:id`

## Uploads
- `POST /api/uploads` com `multipart/form-data`
